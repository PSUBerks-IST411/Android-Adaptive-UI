A Penn State Berks - IST 411 Project (Spring 2014).
This Design & Development project is focused on development of an Android application to enhance accessibility options on smart phones and tablets, so they can be used easily by people with disabilities or anyone else who wants simple and easy way to operate their phones or tablets. This application will also be compatible with NeuroSky MindWave Headset, which uses human mind's activities to control mobile devices. This project will be open source and will be available on GitHub at the end of the timer.
